Bem-vindo ao Keycloak!

Keycloak é uma solução de código aberto para gerenciamento de identidade e acesso. Ele facilita a adição de autenticação e autorização a suas aplicações, oferecendo suporte a Single Sign-On (SSO), LDAP, Active Directory, redes sociais e muito mais.

Vantagens do uso do Keycloak:

  • ✔️ Autenticação e autorização centralizadas
  • ✔️ Suporte a múltiplos protocolos (OpenID Connect, OAuth 2.0, SAML 2.0)
  • ✔️ Integração com LDAP e Active Directory
  • ✔️ Suporte a Single Sign-On (SSO)
  • ✔️ Interface de administração amigável
